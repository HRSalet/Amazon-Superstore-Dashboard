# Amazon Superstore Dashboard

A comprehensive dashboard for analyzing sales, profitability, and operational metrics from Amazon Superstore data.

## 📊 Overview

This dashboard provides an interactive visualizations and insights from a Amazon Superstore data. It enables stakeholders to track Key Performance Indicators (KPIs), identify trends, and make data-driven decisions across sales, customer segments, and product categories.

## 🎯 Key Features

### Performance Metrics
- **Sales Analysis**: Track total sales, sales trends, and year-by-year growth.
- **Profitability Metrics**: Monitor profit margins, profit by city, profit by customer name, profit by product, profit by sub-category.
- **Order Analytics**: Analyze quanities, and order frequency, and return orders.
- **Customer Segmentation**: Analyze and understand customer behavior across different segments (Consumer, Corporate, Home Office).

### Visual Components:
- Interactive charts and graphs for trend analysis.
- Product category breakdowns.
- Area charts showing city-wise performance.

### Filters & Slicers
- Date range selection
- Product category and sub-category filters
- Customer segment filters
